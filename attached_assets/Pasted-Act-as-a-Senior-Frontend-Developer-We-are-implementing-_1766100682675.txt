### FEATURE 2: AUDIOBOOK MODE (HTML Export with Smart Player)
**Goal:** Export a single HTML file that uses the browser's best TTS voices to read the story offline.

**1. UI Changes:**
- Add an "Output Format" toggle/dropdown: [EPUB (Default) | Audiobook (HTML)].

**2. Logic (script.js - Audiobook Generation):**
- Create `generateAudiobookHTML(chapters, metadata)`.
- Combine all chapters into one HTML file.
- **Inject this specific 'Smart Player' script into the footer of the generated HTML:**
  - **Voice Selection Logic:** On load, run `speechSynthesis.getVoices()`. Filter the list to prioritize voices with "Google", "Siri", or "Premium" in their name. Default to these over standard "Microsoft" voices.
  - **The Player UI:** A sticky bottom bar with Play/Pause, Speed (0.8x - 2x), and a "Voice" dropdown so the user can swap voices manually.
  - **Auto-Scroll:** When reading a paragraph, scroll it into view and highlight it (add a CSS class `.reading-now { background: #fdf6e3; }`).
  - **Progress Saving:** Save the current chapter/paragraph index to `localStorage` so they can resume listening if they close the file.

**3. Execution:**
- If "Audiobook" is selected, trigger this function and download `<Title>.html`.
