Act as a Senior Full-Stack Engineer. Build a complete, production-ready application called "FanFicBinder".

Create the following files:
- index.html (Frontend Interface)

- /pages/about.html
- /pages/contact.html
- /pages/privacy.html

2. TECHNICAL REQUIREMENTS
- **Frontend Stack:** HTML5, CSS3, Vanilla JS.
- **Backend Stack:** Netlify Functions (Node.js).
- **Libraries (Frontend CDN):**
  - JSZip: https://unpkg.com/jszip/dist/jszip.min.js
  - EJS: https://unpkg.com/ejs/ejs.min.js
  - jEpub: https://unpkg.com/jepub/dist/jepub.js
- **Libraries (Backend package.json):**
  - axios
  - jsdom
  - @mozilla/readability

3. BACKEND LOGIC (/functions/fetch-url.js)
- Input: Receives a URL query parameter `?url=...`
- Process:
  1. Use `axios` to fetch the HTML of the target URL.
  2. Use `JSDOM` to create a virtual DOM.
  3. Use `@mozilla/readability` to parse the article and strip ads/nav.
  4. Extract: `title`, `byline`, `content` (HTML string).
- Output: Return JSON object with title and content. Handle errors gracefully (400/500 status).

4. FRONTEND LOGIC (script.js)
- **State:** Maintain an array `chapters` to store added content.
- **Input Mode 1: URL Fetcher**
  - Input field for URL.
  - "Fetch Chapter" button calls `/.netlify/functions/fetch-url`.
  - On success, push result to `chapters` array and update UI list.
- **Input Mode 2: Manual Entry**
  - "Add Custom Chapter" button opens a modal/form.
  - Inputs: Chapter Title, Body Text (Textarea).
  - On save, push manual object to `chapters` array.
- **EPUB Generation:**
  - "Download EPUB" button.
  - Initialize `new jEpub()`.
  - Metadata Inputs: Book Title, Author Name.
  - Loop through `chapters` array and add each to the book.
  - Generate `.epub` blob and trigger download.

5. DESIGN SYSTEM (style.css)
- **Theme:** "Writer's Retreat". Background #1c1c1e, Paper/Surface #2c2c2e, Text #e5e5e7, Accent #ff9f0a (Orange).
- **Typography:** 'Merriweather' (Serif) for reading text, 'Inter' for UI.
- **Layout:**
  - Header: Logo "FanFicBinder".
  - Main: Two-column layout.
    - Left: Input Tools (Tabs for "URL" vs "Manual").
    - Right: "Binder Queue" (List of chapters added so far).
- **Components:**
  - Chapter Card: Shows Title + Word Count + "Remove" button.
  - Download Button: Large, prominent, sticky at bottom on mobile.
